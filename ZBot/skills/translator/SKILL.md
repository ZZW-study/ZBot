# 翻译助手

专业的多语言翻译技能，支持自动检测源语言并提供高质量翻译。

## 支持的语言

| 语言 | 代码 |
|------|------|
| 中文 | zh |
| 英文 | en |
| 日文 | ja |
| 韩文 | ko |
| 法文 | fr |
| 德文 | de |
| 西班牙文 | es |
| 俄文 | ru |

## 使用方式

直接告诉我要翻译的内容和目标语言：

```
把这段话翻译成英文：你好，世界
翻译成日语：今天天气真好
这段英文是什么意思：Hello, World!
```

## 翻译原则

1. **准确性**：确保译文准确传达原文含义
2. **流畅性**：译文符合目标语言的表达习惯
3. **专业性**：技术术语保持一致性
4. **本地化**：考虑文化差异，适当调整表达

## 特殊功能

### 术语翻译
遇到专业术语时，会提供：
- 标准翻译
- 常见替代译法
- 保留原文的建议（如 API、SDK 等）

### 润色优化
可以要求对已有译文进行润色：
```
润色这段英文翻译：...
```

### 双语对照
可以要求输出双语对照格式：
```
翻译成英文并显示对照：...
```
